# 为什么要写这两个组件？
> 因为有时候业务需求上展示的内容经常会变，不能直接用展示的内容。所以需要使用一个相对固定、唯一的值应用到代码逻辑里，就像el-select组件那样。
> 原来element ui的RadioGroup、CheckboxGroup组件数据，value和label都是同一个值，无法满足需求，所以想要重新封装两个组件。
